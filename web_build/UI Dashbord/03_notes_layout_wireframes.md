# CryptoBot Pro — Notes: Layout & Wireframes

## 1. Desktop ≥1440px (основной режим)
Сетка: 12 колонок, внутренний max-width ~1440–1600.

### Верхняя часть
- Top-bar во всю ширину.
- Под ним один ряд KPI-карточек (5–6 штук).

### Центральная часть
Две колонки:
- Левая (примерно 8–9 колонок) — график.
- Правая (3–4 колонки) — панель стратегий и бэктеста.

### Нижняя часть
Горизонтальные табы:
- Signals | Positions | Orders | Backtest | Logs
Под табами — содержимое выбранного таба (таблица/метрики).

## 2. 1024–1439px
- График во всю ширину над панелью стратегии.
- Панель стратегий — под графиком в виде аккордеона (сворачиваемые секции).
- KPI-карточки могут идти в 2 ряда.

## 3. 768–1023px (планшет)
- Top-bar в две строки при необходимости.
- KPI — в 2 ряда.
- График — fullscreen по ширине, панель стратегий под ним.
- Табы снизу — с горизонтальным скроллом.

## 4. <768px (мобильный)
- Один столбец.
- Сначала Top-bar, потом KPI (в 2–3 ряда по 2 карточки).
- График — затем аккордеон “Strategy & Backtest”.
- Табы — внизу, тоже как аккордеон/скролл.

## 5. Важные UX-моменты
- KPI никогда не прячем глубоко; максимум — во второй строке.
- Правую шкалу используем только для цены, левую — для equity.
- Цветовая кодировка сделок:
  - BUY — зелёный
  - SELL — красный
  - SL — красная пунктирная линия
  - TP — зелёная пунктирная линия
- Не перегружаем панель стратегий — важные параметры ближе к верху.
