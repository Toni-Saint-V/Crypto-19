# CryptoBot Pro — UI Handoff Package

В этом архиве лежат материалы для реализации дашборда CryptoBot Pro.

## Содержимое
- `01_senior_dev_instructions.md` — подробная инструкция для старшего разработчика.
- `02_ui_dashboard_brief_original.md` — исходный UX/UI бриф.
- `03_notes_layout_wireframes.md` — текстовое описание лейаутов и вайрфреймов.
- `design_tokens.css` — CSS-переменные для темы и типографики.

Рекомендуемый порядок изучения:
1. Прочитать `02_ui_dashboard_brief_original.md`, чтобы понять vision и требования.
2. Пройтись по `01_senior_dev_instructions.md` — там описано, как это лучше реализовать технически.
3. Открыть `03_notes_layout_wireframes.md` — использовать как подсказку при верстке лейаута.
4. Подключить `design_tokens.css` и дальше уже строить компоненты.

Если нужно расширить пакет — добавить сюда описание API, схемы данных или скриншоты из Figma.
