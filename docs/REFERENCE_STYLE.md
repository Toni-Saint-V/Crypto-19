# Reference Style (скрин "Bot Analytics Dashboard" с ноутбуком и телефоном)

Цель: UI выглядит как премиальный dark-neon analytics терминал.

## Визуальные признаки референса
- Темная палитра с мягкими градиентами (background).
- Неон-акцент (синий/фиолетовый) для активных элементов и линий.
- Фоновые "волны/линии" разных цветов (тонкие кривые), создают глубину.
- Большой заголовок (hero typography) и аккуратный glow.
- Компоненты "glass" (полупрозрачные карточки), мягкие тени.
- Скругления крупнее стандартных, единый радиус по системе.
- Плотная сетка: много данных, но читабельно.
- Визуал девайсов (ноутбук + телефон) как ключевой герой референса.

## Как перенести в продуктовый дашборд (внутри приложения)
- В верхней зоне над графиком сделать "hero strip":
  - слева: заголовок/контекст (кратко), режим, символ, таймфрейм, статус
  - справа: компактный "device hero card" с иллюстрацией телефона (как из референса), с мягким glow
- Фоновые волны реализовать как легкий SVG/градиент на фоне layout (не мешает контенту).
