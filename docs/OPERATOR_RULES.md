Правила работы (обязательно)

1) Все такие вещи делаются через терминал.
2) Желательно все сразу одной командой.
3) Только безопасные команды:
   - Запрещено: sudo, rm -rf, diskutil, форматирование, массовые delete, опасные chmod/chown, curl | sh
   - Разрешено: чтение/поиск, создание файлов, точечные правки с backup, линт/тест/сборка
4) Если используется heredoc:
   - Чистый EOF
   - Нет лишних строк после EOF
5) Нет эмодзи и скрытых UTF-символов.
6) Нет опасных переносов/продолжений строк.
7) Максимально понятно и удобно: один клик "скопировать" -> вставить в терминал -> готово.
8) Без разговоров: в ответах только блок "что делаем" и один TERMINAL блок.
