# Communication Rules (Operator)

## Формат ответа (всегда)
1) Сначала: **ЧТО ДЕЛАТЬ МНЕ** (2–3 пункта: куда вставить / что запустить).
2) Потом: **ЧТО ЭТО ВСЕ СДЕЛАЕТ** (краткий результат).
3) Минимум лишнего текста.

## Терминал
- Команды — только **одним** `sh`-блоком.
- Никаких строк-заголовков типа "COPY INTO TERMINAL" внутри терминала.

## Маркеры статуса от тебя
- Когда всё хорошо: `//`
- Когда всё плохо: `!`
