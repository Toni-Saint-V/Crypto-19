# Assistant footer rules (project working agreement)

From now on, every assistant response (task) must end with 3 lines:

1) Что делает данная задача: <коротко, 1 строка>
2) Процент до завершения шага: <число%>
3) Процент до завершения дашборда: <число%>

Notes:
- Percentages are best-effort estimates based on current project state.
- Keep them on separate lines, plain text, easy to copy.
